##  全局安装pic2sprite
```
npm install -g pic2sprite
```

##  pic2sprite
```
pic2sprite 
```

##  pic2sprite 生成 rem 单位的css文件
```
pic2sprite -u
```

#### 合并当前目录下面的所有jpg和png图片  
#### 并生成对于的css背景文件，支持 rem
- u：rem 输出比例，默认 100
- m：提出background为公共样式，background-position单独输出
- n：背景图名称
- b: 小于 b kb用base64替换，默认为 6
